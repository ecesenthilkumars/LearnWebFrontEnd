HTML Notes

Character Escape - To use special charachters in the content

< - &lt; - to use the less than symbol in the content (in html this means open of tag)

> - &gt; - to use the greater than symbol in the content (in html this means close of tag)

& - &amp; - to use the

© - &copy; - To use for copyright symbol &copy;

Non-breaking space - &nbsp; - To make a set of words to appear together

" - &quot; - Some browsers/ encoding might not recognise "", so " shall be used

Links in HTML


<a href="" target="" name="">.....</a>

Anchor tag is used for hyperlinking

Three types of linking possible,

-> Internal file linking

-> internal content linking

-> external linking

<a> </a> can act as both inline element and block level element.


Internal Links:

Link to files from the same directory

<a href="link-to-file.extension" title="same dir link">……</a>


External Links:

Link to any URI, 

<a href="https://targetURL" target="_blank" name="…">someContentToShowAsLink</a>

target="_blank" will instruct browser to open the link in new tab.


Internal content linking:

<a href="#NameOrId">LinkText</a>

# - indicates internal linking within the same document.
